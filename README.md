# Match Lab

Match Lab is a personal environment to do experiments with various programming languages and frameworks in a "near" real-world use-case scenario: making HTTP calls, do some CRUD, experiments with Frameworks and architecture.

This will be a monorepo: I will add frontend and backend folders, and inside of them I will replicate the same application using different languages and framework.

Here in the root of the Project I might add some resources to build up the various versions of the application. I will think about this when something "working" will come up.
